# 書道教室予約ホームページ
PHP自作

## 概要
書道教室のホームページにレッスンの予約機能をつけました。
カレンダー機能を追加し、予約の残席数を表示しております。

管理ユーザと一般ユーザに分け、
それぞれでログインできるようにしています。

## 使い方
管理ユーザ

予約者の一覧が見れます。
レッスンの新規作成、削除が可能です。
